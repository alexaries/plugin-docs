## 说明
采用git-book+markdown编写文档

### 安装
- win
```
npm install gitbook -g
npm install gitbook-cli -g
npm install gh-pages -g
```

- mac
```
sudo npm install gitbook -g
sudo npm install gitbook-cli -g
sudo npm install gh-pages -g
```

## 构建
```
gitbook build
```
## 运行 
```
gitbook serve
```
